# Palworld 右鍵連點器

一個用於《幻獸帕魯》（Palworld）的簡易右鍵連點工具。主要功能是在**按下滑鼠右鍵時，自動連續觸發 10 次右鍵點擊**，方便玩家在建築模式下快速連續放置 10 個建築物。

## ✨ 功能特點

*   右鍵觸發後自動執行 10 次右鍵點擊
*   適合建築連續放置，節省手動點擊時間
*   輕量、簡單、免安裝

## 📦 使用方式

1.  **啟動程式**：下載並執行 `auto_clicker.exe`。
2.  **進入遊戲**：在遊戲中進入建築模式。
3.  **觸發連點**：按下滑鼠右鍵。
4.  **完成**：程式會幫你自動連續點擊 10 次，快速放置 10 個建築。
