# 《幻獸帕魯》（Palworld）左鍵連點器

一個用於《幻獸帕魯》（Palworld）的簡易左鍵連點工具。主要功能是通過**熱鍵觸發，自動連續執行多次左鍵點擊**（預設 10 次），方便玩家在建築模式下快速連續放置建築物。

## ✨ 功能特點

*   通過熱鍵觸發左鍵連點（預設 F2 鍵）
*   可自訂連點次數（1-100 次，預設 10 次）
*   可自訂觸發熱鍵（F1-F12）
*   總開關控制啟動/停止偵測
*   輕量、簡單、免安裝
*   設定自動儲存

## 📦 使用方式

1.  **下載並啟動程式**：從 Releases 下載 `auto_clicker.exe` 並執行。
2.  **設定參數**（可選）：
   - 選擇觸發熱鍵（F1-F12）
   - 設定連點次數（1-100）
3.  **進入遊戲**：在《幻獸帕魯》中進入建築模式。
4.  **觸發連點**：按下設定的熱鍵（預設 F2）。
5.  **完成**：程式會自動連續執行左鍵點擊，快速放置建築物。
